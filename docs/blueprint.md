# **App Name**: LocalBid Exchange

## Core Features:

- Verified User Onboarding: Users can register and log in, with a streamlined process including OTP verification for profile creation and general verification to ensure a trusted community.
- Item Listing & Discovery: Sellers can easily post detailed listings with images and descriptions, while buyers can browse, search, and filter products by category and location.
- Secure Transaction System: An integrated payment gateway facilitates purchases through an escrow system, automatically applying the 8-10% commission and handling all financial flows securely.
- In-App Messaging & Meeting Suggestions: Secure direct messaging between buyers and sellers, coupled with smart suggestions for safe, public meeting points based on their agreed locations.
- Seller Reliability Ratings & Reviews: Buyers can rate and leave reviews for sellers post-transaction, contributing to a visible seller profile displaying total sales and an aggregate reliability score.
- Timed Auction Functionality: Sellers can list items for auction with a specified duration and starting bid, allowing buyers to place competitive bids within the given timeframe.
- AI Demand Spotlighting Tool: An AI-powered tool analyzes listing views, user searches, and past transaction data to suggest high-demand areas or optimal listing categories, visually represented to guide sellers.
- Listing Boosts: Sellers can pay to boost their listings, increasing visibility and reach within the app.
- ID Verification Badge: Users can opt to undergo an ID verification process to earn a 'Verified' badge on their profile, enhancing trust.
- Anti-Scam Chat Filters: An AI-powered tool automatically scans and filters chat messages for suspicious links, phone numbers, and common scam phrases, preventing fraudulent interactions.
- User Reporting and Blocking: Users can report suspicious activity or inappropriate content from other users and block them, contributing to a safer community.
- Location-based Discovery Filters: Buyers can refine their product search by proximity, filtering listings within a 5km, 10km, or 25km radius from their current location.

## Style Guidelines:

- Primary color: A solid, dependable blue (#225BC3) representing trust and efficiency, serving as a prominent color for interactive elements and branding.
- Background color: A very light, clean blue-grey (#EEF1F3) that provides an airy, spacious feel, ensuring readability and focus on content.
- Accent color: A vibrant aqua (#34CBED) to highlight important calls to action, notifications, and interactive features, bringing a touch of modernity and energy.
- Headline and body font: 'Inter', a grotesque-style sans-serif, chosen for its modern, neutral, and highly readable qualities across all text elements.
- Use a set of clear, concise line-art icons that complement the clean interface, focusing on ease of understanding for marketplace actions like listing, buying, and messaging.
- Employ a modular, card-based layout for product listings, promoting scannability and intuitive organization, with clear visual hierarchy for essential information.
- Implement subtle transition animations for page changes and dynamic elements, providing smooth user feedback without being distracting, enhancing the sense of polish.
